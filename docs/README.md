## One Important Rule

Write down how a feature should work before changing its code. Use the
[feature research template](research/FEATURE_TEMPLATE.md). This makes review
easier and prevents a confident-looking guess from becoming game behavior.

## Words Used in These Guides

- **Apex** means the original Apex Legends game.
- **Lapex** means this Minecraft recreation.
- **Exact** means the visible rule works the same way within Minecraft's limits.
- **Adapted** means the same combat idea uses a Minecraft-friendly rule.
- **Analogue** means Minecraft has no matching system, so Lapex uses a clear substitute.
- **Not implemented** means the feature is not in the game yet. It must not be advertised as working.
