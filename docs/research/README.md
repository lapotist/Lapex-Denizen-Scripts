# Feature Research Notes

This folder holds the design evidence for gameplay work.

Before code is added, copy `FEATURE_TEMPLATE.md` into `features/` and fill it
out. A small change may use a short note. A legend rework should document every
passive, tactical, ultimate, interaction, cooldown, range, charge, and failure
state that affects players.

Research notes are living records. When a new Apex patch changes a rule, add the
new source and mark the old statement as replaced. Do not silently rewrite the
history.
